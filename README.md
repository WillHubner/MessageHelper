# MessageHelper

Popup message helper to FMX Forms.

### How to use

```delphi
Boss install https://github.com/WillHubner/MessageHelper
```

Declare Form.Helper.Message on destiny form.

```delphi
Mensagem('Ok message!', tmOK);
Mensagem('Error message!', tmErro);
```
